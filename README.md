# Spider_ML_Task_2

## Contents:
- Task 1: Create a ResNet-CNN on the CIFAR-10 dataset based on the given constraints.
- Task 2 and 3: Do a comparative analysis of temperature forecasting from weather input data on the Jena Climate Dataset between a from-scratch LSTM and Transformer based on the given constarints.
- Applied Domain Task: Create a medical RAG pipeline chatbot that ingests the given medical guidelines from WHO, NICE etc as well as the MedQuad Dataset to give grounded and accurate responses to health queries, while keeping safety and warning guardrails and giving advice to consult a professional if required. A confidence score for the answer has also been calculated along with evaluation metrics that include retrieval accuracy and hallucination risk measure based on an NLI cross-encoder.
